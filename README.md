My Student Life Portfolio Website
This personal portfolio website showcases my journey and experiences as a BSIT student at POC-NEUST.
It is designed to visually represent my academic life, interests, and the skills I have developed throughout my college years.

Technologies Used:
This project was built using the following technologies as required:
Java,Spring Boot,Thymeleaf,HTML5 & CSS 

Main Pages	
The website consists of the following five core pages:
Home Page (/)-Features a short welcome message, my full name, course, year level, and a profile photo.
About Me (/about)-Includes a brief biography, my goals as an IT student, my interests, and why I chose BSIT.
My College Journey (/journey)-Describes my memorable class experiences, challenges I've faced, and lessons learned so far.
Skills and Interests (/skills)-Highlights my programming skills (such as Java, HTML, and CSS) and other personal hobbies.
Photo Gallery (/gallery)-A collection of moments from my student life with descriptive captions.

Project Structure
src/main/java-Contains the Spring Boot controllers.
src/main/resources/templates-The Thymeleaf HTML templates.
src/main/resources/static- Stores static assets such as CSS files and project images.

#SHORT REFLECTION
Creating my student life portfolio using Spring Boot and Thymeleaf was a fun and meaningful experience for me.
The part I enjoyed the most was adding images to my pages. Seeing those pictures felt special because they represent
our real experiences and memorable moments as BSIT students. I also liked the design process a lot.
Designing has always been one of my hobbies, so being able to use my creativity in a school project made the task more
enjoyable and natural for me.

Through this activity, I learned many technical skills. I improved my knowledge in Java, HTML, and CSS,
and I learned how to build a well-structured website. I also understood how to create routes so users can easily move between
pages like Home, About, and Journey. Aside from technical skills, this project helped me become more creative.
I realized that coding is not only about logic and functions, but also about presenting ideas and stories in a clear and interesting way.

This portfolio shows that I am an organized student. I made sure that the layout is consistent and the files are properly arranged.
It reflects my effort, dedication, and passion for the BSIT course. Overall, this portfolio represents my hard work,
my interests, and my journey as I continue to grow and prepare for the future.
